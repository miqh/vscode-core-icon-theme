# Changelog

## [0.2.0] - 2025-12-23

Pre-release version due to significant project scaffolding changes.

- **Breaking:** Remove icon theme variants that were exposed for each icon design.
- Add new extension-specific settings for customising the icon theme.
- Add a web extension target so a limited version of the icon theme can be used
  in a web-based contexts.
- Add new "Round Bar" icon design option.

## [0.1.0] - 2025-05-04

Initial release.
